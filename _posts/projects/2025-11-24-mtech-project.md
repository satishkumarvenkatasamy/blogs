## Abstract

Traditional hiring processes in the Information Technology sector are time- consuming and resource-intensive, requiring substantial effort from hiring man- agers, developers, and recruiters. This research presents AI Recruiter 2.0, an intelligent hiring automation system that leverages knowledge graph-based code analysis to provide comprehensive, multi-dimensional candidate evaluations. Unlike conventional Applicant Tracking Systems that rely on keyword matching, our approach constructs semantic knowledge graphs from candidates’ GitHub repositories, enabling deep analysis of code architecture, quality, security, and documentation. The system integrates five state-of-the-art Large Language Mod- els (OpenAI GPT-4, Anthropic Claude, Google Gemini, Mistral, and xAI Grok) in an ensemble framework to ensure robust, bias-free assessments. A novel math- ematical framework derives quantitative quality metrics from graph-theoretic operations on code structures. Evaluation on 44 repositories (14 controlled test cases and 30 real-world projects) demonstrates 92.9% F1-score accuracy in code quality assessment and 90% reduction in screening time. Critically, the system generates explainable, auditable scoring that enables organizations to demonstrate compliance with AI governance regulations and anti-discrimination mandates, supporting fair and transparent hiring decisions.
